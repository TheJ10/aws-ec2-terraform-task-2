# AWS EC2 Deployment – Manual & Terraform

## Overview
This repository documents my DevOps internship task focused on understanding AWS core concepts and Terraform by deploying an EC2 instance manually and using Infrastructure as Code (IaC).

---

## Objectives
- Learn AWS core concepts
- Understand Infrastructure as Code (IaC)
- Launch an EC2 instance manually using AWS Console
- Provision an EC2 instance using Terraform
- Follow DevOps best practices with proper documentation and version control
